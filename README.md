## Progression

| Level | Emne | Du lærer at... |
|-------|------|----------------|
| 1     | Classes and Objects Basics | Skabe objekter med instance variables og constructors |
| 2     | Instance Methods | Lade objekter udføre handlinger på sig selv |
| 3     | Object Interaction | Få objekter til at kommunikere og samarbejde |
| 4     | Information Hiding, toString og Static | Beskytte data med access modifiers, printe objekter læsbart, og dele data mellem objekter |
## Sådan bruger du dette materiale

Dette materiale består af **concepts** (forklaringer) og **assignments** (træningsopgaver) der hører sammen.

### Forberedelse før lektion

1. Læs det relevante **concept** dokument igennem
2. Lav de tilhørende **assignments** – enten som en del af forberedelsen eller efter lektionen

Opgaverne er designet til at træne de specifikke færdigheder fra hvert level. De er korte, fokuserede og har hints hvis du går i stå.

### Ugeopgaver (torsdage)

Udover træningsopgaverne her, er der også et ugentligt opgavesæt som laves om torsdagen. Disse **ugeopgaver** er:

- Mere åbne og komplekse
- Kombinerer flere emner
- Kræver selvstændig problemløsning
- Afleveres og får feedback

Træningsopgaverne her forbereder dig til ugeopgaverne.

### Iterativt projekt

[SP1](https://1sem.kursusmaterialer.dk/projects/SP1/) projektet følger dig gennem hele Flow1.
Du har måske allerede arbejdet med variabler og kontrolstrukturer. I denne fase kan du nu organisere din kode med metoder og objekter.
Dette vil gøre dit projekt mere struktureret, læsbart og nemmere at udvide.
